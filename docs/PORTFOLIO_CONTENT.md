# Portfolio Content

## Project summary
AI Interview Coach is a full-stack application that helps software engineers prepare for interviews by generating resume-aware questions, evaluating answers, and delivering personalized study plans. It demonstrates modern backend, mobile/web frontend, AI integration, and production readiness.

## What I built
- Full FastAPI backend with auth, resume upload, interview sessions, analytics, and study plan generation.
- Expo React Native frontend for cross-platform interview practice and performance tracking.
- Resume parsing with PyMuPDF and Gemini-powered question generation.
- Docker and GitHub Actions CI to support reproducible builds and deployment.

## Skills demonstrated
- Backend API design and security
- Database modeling and migrations
- LLM integration and prompt engineering
- Cross-platform mobile/web UI development
- DevOps readiness with Docker and CI/CD
- Technical writing and documentation

This project shows my ability to produce a complete product from concept to deployable artifact. It illustrates practical AI application work, solid software architecture, and a disciplined approach to quality through testing and documentation.
