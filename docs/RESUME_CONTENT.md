# Resume Content

## A. One-line version
AI Interview Coach: a full-stack AI interview preparation platform with FastAPI backend, Expo React Native frontend, Gemini-powered question generation and answer evaluation, and analytics-driven study plans.

## B. Two-line version
Developed an AI Interview Coach that ingests resumes, generates company-specific interview questions, evaluates responses, and provides personalized study plans. Built with FastAPI, SQLAlchemy, PyMuPDF, Expo React Native, and Gemini, with Dockerized deployment readiness.

## C. Detailed 5-line version
- Built a full-stack interview preparation MVP featuring secure user registration, resume upload, and resume parsing using PyMuPDF.
- Implemented a FastAPI backend that generates tailored interview questions and follow-ups with Gemini integration while preserving deterministic offline fallbacks.
- Added answer evaluation, analytics, and study-plan engines to identify strengths, weakness trends, and guided review topics.
- Created a React Native Expo frontend with auth flow, resume upload, interactive interview session screens, and analytics dashboards.
- Delivered production-ready deployment artifacts including Docker support, GitHub Actions CI, environment configuration, and technical documentation.

## D. Technologies used
- FastAPI, Python, SQLAlchemy, Alembic
- PyMuPDF, Gemini API (`google-generativeai`)
- SQLite (dev), Docker, GitHub Actions
- Expo React Native, TypeScript, Axios, AsyncStorage
- black, isort, flake8, pytest, pytest-cov, pip-audit

## E. Key achievements
- Built end-to-end AI interview preparation platform from backend to mobile/web UI.
- Enabled resume-aware question generation and automated answer evaluation using Gemini with fallback reliability.
- Designed analytics and study plan engines that track weaknesses and create personalized learning plans.
- Added CI/CD automation, production deployment readiness, and developer documentation for portfolio presentation.
