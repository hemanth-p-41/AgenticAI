# Achievements

## Project accomplishments
- Delivered a full-stack AI interview coach with resume-aware question generation, answer evaluation, analytics, and study plan creation.
- Integrated Gemini with a robust service layer and deterministic fallback mode for stable development and testing.
- Built secure backend architecture with FastAPI, SQLAlchemy, JWT auth, file upload handling, and production middleware.
- Created a React Native Expo frontend for cross-platform interview preparation and analytics access.
- Added production artifacts including Docker support, GitHub Actions CI workflows, and developer documentation.

## Technical highlights
- Designed an interview session model that stores questions, user responses, and evaluation scores for longitudinal coaching.
- Implemented resume parsing using PyMuPDF to ground AI-generated questions in candidate experience.
- Built analytics and study plan engines to convert performance data into actionable review plans.
- Maintained stable automation by separating Gemini prompt logic from fallback behavior.

## Real-world value
- Empowered candidates to practice relevant interview questions tied to their resume.
- Reduced preparation time by focusing on weak areas identified through analytics.
- Made the codebase deployable and maintainable with CI and Docker, ready for extension into production.
